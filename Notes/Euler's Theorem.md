``` ad-Theorem
title: Theorem (Euler).

Fix $n\in\N^+$. For all $a\perp n$, we have $a^{\phi\l(n\r)}\equiv1\mod{n}$.

```

<i>Proof.</i> Note that $a\in\Z_n^\times$, so, by Lagrangeâ€™s Theorem, we see that $\ord{a}$ divides $\ord{\Z_n^\times}=\phi\l(n\r)$. Hence $a^{\phi\l(n\r)}\equiv1\mod{n}$, as desired.<span style="float:right;">$\blacksquare$</span>
