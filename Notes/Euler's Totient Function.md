``` ad-Definition
title: Definition.

_**Euler$\textrm{'}$s Totient Function** is the function $\phi:\N^+\to\N^+$ defined by $\phi\l(n\r)\coloneqq\ord{\Z_n^\times}$ for all $n\in\N^+$; that is_
$$\begin{equation}
    \phi\l(n\r)\coloneqq\l|\l\{1\leq m\leq n\mid m\perp n\r\}\r|.
\end{equation}$$
```
