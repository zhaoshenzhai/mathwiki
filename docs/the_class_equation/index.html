The Class Equation

Statement & Proof

A group $G$ acting on a finite set $X$ naturally partitions $X$ into its orbits $X=\bigsqcup_{i\in I}Gx_i$, which can either be trivial or non-trivial. The trivial ones are singletons $\l\{x_i\r\}$. We call those the fixed points of the action, denoted $Z\coloneqq\l\{x\in X\st gx=x\textrm{ for all }g\in G\r\}$. If some structure is known about the non-trivial orbits, then this partition allows us to ‘glean’ information on $\l|Z\r|$.


  More precisely, we have the following relation between $|X|$ and $|Z|$.

Theorem (The Class Equation). Let $G\act X$ act on a finite set $X$. Then $$\begin{equation} \l|X\r|=\l|Z\r|+\sum_{a\in A}\l[G:G_a\r], \end{equation}$$ where $A\subseteq X$ contain exactly the elements $a\in X$ with $\l|Ga\r|>1$.
Proof.

Since we have a partition $X=\bigsqcup_{i\in I}Gx_i$ and $\l|Gx\r|=1$ for all $x\in Z$, so splitting off the two cases gives us $$\begin{equation} \l|X\r|=\l|Z\r|+\sum_{a\in A}\l|Ga\r|=\l|Z\r|+\sum_{a\in A}\l|G\r|/\l|G_a\r|, \end{equation}$$ where the last equality follows from Orbit-Stabilizer. By Lagrange’s Theorem, we have $[G:G_a]=\l|G\r|/\l|G_a\r|$ for each such $a$.$\blacksquare$

Conjugation Action

Let $G$ be a finite group, acting on itself by conjugation via $\phi$. With $Z=Z(G)\coloneqq\ker\phi$ the center of $G$ and with $Z_G(a)$ the stabilizer of $a$, we have the classical Class Equation, stating that $$\begin{equation} \l|G\r|=\l|Z(G)\r|+\sum_{a\in A}[G:Z_G(a)], \end{equation}$$ where $A\subseteq G$ contain exactly the elements with non-trivial conjugacy class.


  This can sometimes be used to show that $Z(G)$ is non-trivial, giving us a fixed-point theorem. See, for instance, how it applies$^\dagger$ to $p$-groups.
