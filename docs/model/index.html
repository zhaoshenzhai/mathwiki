Tarskian Definition

\begin{definition}

Let $L\coloneqq(\mc{F},\mc{R},\mf{a})$ be a \ref[first-order language]{language}. An $L$-structure is a tuple $\mc{M}\coloneqq(M,i)$ consisting of a set $M$, called the universe, and an interpretation function $i$ assigning

  • to each function symbol $f\in\mc{F}$, an $\mf{a}(f)$-ary function $f^\mc{M}:M^{\mf{a}(f)}\to M$, and
  • to each relation symbol $R\in\mc{F}$, an $\mf{a}(R)$-ary relation $R^\mc{M}\subeq M^{\mf{a}(R)}$.

\end{definition}

Interpretation

Let $\mc{M}$ be an $L$-structure. We define the interpretation of $L$-terms and $L$-formulas in $\mc{M}$.

\begin{definition}

The interpretation of an $L$-term $\tau(\bar{x})$ in $\mc{M}$ is the function $\tau^\mc{M}:M^{|\bar{x}|}\to M$ defined by:

  • If $\tau=c\in\mc{C}$, then $\tau^\mc{M}(\bar{a})\coloneqq c^\mc{M}$ for all $\bar{a}\in M^{|\bar{x}|}$.
  • If $\tau=x$ is a variable, then $x=x_i$ for some $1\leq i\leq n$, and let $\tau^\mc{M}(\bar{a})\coloneqq a_i$ for all $\bar{a}\in M^{|\bar{x}|}$.
  • If $\tau=f(\tau_1,\dots,\tau_n)$ for some $f\in\mc{F}_n$, then $\tau^\mc{M}(\bar{a})\coloneqq f^\mc{M}(\tau_1^\mc{M}(\bar{a}),\dots,\tau_n^\mc{M}(\bar{a}))$ for all $\bar{a}\in M^{|\bar{x}|}$.

\end{definition}

\begin{definition}

The interpretation of an $L$-formula $\phi(\bar{x})$ in $\mc{M}$ is the relation $\phi^\mc{M}\subeq M^{|\bar{x}|}$ defined by:

  • If $\phi=(\tau_1=\tau_2)$, then $\phi^\mc{M}\coloneqq\{\bar{a}\in M^{|\bar{x}|}\st\tau_1^\mc{M}(\bar{a})=\tau_2^\mc{M}(\bar{a})\}$.
  • If $\phi=R(\tau_1,\dots,\tau_n)$ for some $R\in\mc{R}_n$, then $\phi^\mc{M}\coloneqq\{\bar{a}\in M^{|\bar{x}|}\st(\tau_1^\mc{M}(\bar{a}),\dots,\tau_n^\mc{M}(\bar{a}))\in R^\mc{M}\}$.
  • If $\phi=\phi_1\circ\phi_2$ where $\circ\in\l\{\land,\lor,\rightarrow,\leftrightarrow\r\}$, then $\phi^\mc{M}\coloneqq\{\bar{a}\in M^{|\bar{x}|}\st\bar{a}\in\phi_1^\mc{M}\circ\bar{a}\in\phi_2^\mc{M}\}$.
  • If $\phi=\lnot\psi$, then $\phi^\mc{M}\coloneqq\{\bar{a}\in M^{|\bar{x}|}\st\lnot\bar{a}\in\psi^\mc{M}\}$.
  • If $\phi=\ex y\,\psi$, then $\psi=\psi(\bar{x},y)$, and $\phi^\mc{M}\coloneqq\{\bar{a}\in M^{|\bar{x}|}\st\ex b\in M((\bar{a},b)\in\psi^\mc{M})\}$.

If $\bar{a}\in\phi^M$, we write $M\models\phi(\bar{a})$ instead, and say that $M$ models $\phi(\bar{a})$. The notation $\phi(\mc{M})\coloneqq\phi^\mc{M}$ is also used.

\end{definition}

Given $A\subeq M$, the $A$-expansion of $L$ is the language $L_A\coloneqq L\sqcup\{\underline{a}\}_{a\in A}$, where $\underline{a}$ is a new constant symbol.

Theory

Let $\mc{M}$ be an $L$-structure. The theory of $M$ is the $L$-theory $\Th\mc{M}\coloneqq\l\{\sigma\in\Sent(L)\st M\models\sigma\r\}$. Two $L$-structures $\mc{M}$ and $\mc{N}$ are elementarily equivalent if $\Th\mc{M}=\Th\mc{N}$, in which case we write $\mc{M}\equiv\mc{N}$.

Theories and Elementarity

A class $\mc{K}$ of $L$-structures is an elementary class if there is an $L$-theory $T$ such that $\mc{K}=\l\{\mc{M}\st\mc{M}\models T\r\}$.
