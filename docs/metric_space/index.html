Metric Space

Abstracting away the basic properties of Euclidean distance in $\R^n$, and most notably preserving the Triangle Inequality, we make the following

Definition.

A metric on a set $X$ is a function $d:X\times X\to\R_{\geq0}$ such that

  1. (Positive-definiteness). For all $x,y\in X$, we have $d(x,y)=0$ iff $x=y$.
  2. (Symmetry.) For all $x,y\in X$, we have $d(x,y)=d(y,x)$.
  3. ($\Delta$-inequality). For all $x,y,z\in X$, we have $d(x,z)\leq d(x,y)+d(y,z)$.

If $d$ is a metric on $X$, we call the pair $\tpl{X,d}$ a metric space.

Let $\tpl{X,d}$ be a metric space. The most important objects that the metric $d$ construct are open balls, which form $^\dagger$ a basis for a topology. Explicitly, this means that a subset $U\subseteq X$ is open if for every $x\in U$, there is some $\epsilon>0$ such that $B_\epsilon(x)\subseteq U$, where

Definition. For any $x\in X$ and $\epsilon>0$, the $\epsilon$-ball centered at $x$ is the set $$\begin{equation} B_\epsilon(x)\coloneqq B(\epsilon,x)\coloneqq\l\{y\in X\st d(x,y)<\epsilon\r\}. \end{equation}$$

In this way, many concepts that were previously defined for metric spaces are now subsumed by their topological characterizations. Nonetheless, metric spaces are still an important class of topological spaces, which is made precise by the following

Definition. A topological space $\tpl{X,\mc{T}}$ is said to be metrizable if there is a metric $d$ on $X$ whose open balls generate $\mc{T}$.

Clearly, metric spaces with continuous maps form a subcategory of $\cattop$. However, there is a more useful subcategory in which morphisms preserve the metric structure.

Definition. Fix $k\in\N$. A function $f:X\to Y$ is said to be $k$-Lipschitz if $$\begin{equation} d(f(x),f(x’))\leq kd(x,x’) \end{equation}$$ for all $x,x’\in X$. We say that $f$ is Lipschitz if it is $k$-Lipschitz for some $k$.

We let $\catmet$ be the category whose objects are metric spaces and whose morphisms are $1$-Lipschitz maps (also called non-expanding maps). Since Lipschitz maps are continuous, this makes $\catmet$ a subcategory of $\cattop$.
