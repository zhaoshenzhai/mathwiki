Universal Arrows

Definition.

Let $F:D\to C$ be a functor and fix $c\in C$. A universal arrow from $c$ to $F$ is an initial object in $(c\downto F)$. That is, it is a pair $(r,u)$, with $r\in D$ and $u:c\to Fr$, inducing for each $d\in D$ a bijection $$\begin{equation} \phi_d:D(r,d)\to C(c,Fd):f’\mapsto Ff’\circ u. \end{equation}$$ This can be visualized as in the following diagram.

Remark. The bijection $\phi_d$ is natural in $d$, for if $g’:d\to d’$, then $$\begin{equation} F(g’\circ f’)\circ u=Fg’\circ(Ff’\circ u) \end{equation}$$ for every morphism $f’:r\to d$, making its naturality square commute. Conversely, by the Yoneda Lemma, every natural isomorphism $\phi:D(r,\slot)\to C(c,F\slot)$ arises in this way via a unique universal arrow $u:c\to Fr$ from $c$ to $F$.$\blacklozenge$
