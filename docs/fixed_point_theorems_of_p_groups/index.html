Fixed-point Theorems for $p\ $-groups

Non-triviality of $Z(G)$

A standard method to study a finite group is to first find its center; if non-trivial, we can set up an induction proof for certain existence theorems by quotienting by $Z(G)$.


  This is indeed the case for finite $p$-groups, which we now show. Throughout, let $G$ be a non-trivial finite $p$-group.

Lemma.

If $G$

acts on a finite set $S$, then $\l|Z\r|\equiv\l|S\r|\,\mod\,p$ where $Z$ is the set of fixed-points of this action.

Proof.
Click to expand...

Letting $A\subseteq S$ contain exactly the elements with non-trivial orbits, we have $$\begin{equation} \l|S\r|=\l|Z\r|+\sum_{a\in A}\l[G:G_a\r] \end{equation}$$ by the

Class Equation. Since $G$ is a $p$-group, each summand $\l[G:G_a\r]$ is a power of $p$ greater than $1$, and hence is $0\,\mod\,p$.$\blacksquare$
Proposition. The center $Z(G)$ is nontrivial.
Proof.
Click to expand...
Letting $G$ act on itself by conjugation, the above gives us $\l|Z(G)\r|\equiv\l|G\r|\,\mod\,p$. Since $\l|G\r|>1$ is a power of $p$, this shows that $\l|Z(G)\r|$ is a multiple of $p$. As $e\in Z(G)$, we see that $\l|Z(G)\r|\neq0$ and hence $\l|Z(G)\r|\geq p$.$\blacksquare$

More generally, we have the following

Proposition.

If $H\nsubgrpeq G$, then $H\cap Z(G)$ is non-trivial.


  Furthermore, if $\l|H\r|=p$, then $H\substructeq Z(G)$.

Proof.
Click to expand...

Same as above, by letting $G$ act on $H$ by conjugation.


  If $\l|H\r|=p$, then every non-trivial $x\in H\cap Z(G)$ generates

$^\dagger$ $H=\gen{x}\substructeq Z(G)$.$\blacksquare$
