Order

Motivation & Definition

Let $G$ be a group. The order of an element $x\in G$ is the size of its generated cyclic subgroup. A more elementary definition is as follows.

Definition.

An element $x\in G$ has finite order if $x^n=e$ for some $n>0$; the least such $n$ is the order of $x$, denoted $\ord x$.


  Otherwise, set $\ord x\coloneqq\infty$, and say that $x$ has infinite order.

Proof ($\ord x=|\!\gen{x}|$). Click to expand... Since $\gen{x}$ is the image of the map $\phi_x:\Z\to G$ induced uniquely by sending $1\mapsto x$, if $\ker\phi_x=n\Z$ for some $n>0$, then$^\dagger$ $\im\phi_x\iso\Z/n\Z$ has cardinality $n$. Those are the only$^\dagger$ non-trivial subgroups of $\Z$, so otherwise, we have $\im\phi_x=\Z$.$\blacksquare$

Examples

If $G$ is cyclic, then the order of any generator is $|G|$. In general:

  • If $G$ is finite, then by Lagrange’s Theorem, the order of any $x\in G$ divides $|G|$. In particular, if $p\coloneqq|G|$ is prime, then $G\iso\Z/p\Z$.
  • Since $\Z$ is a UFD, the order of every $x\in G$ has a prime factorization. If there is a prime $p$ such that every $\ord x$ is a power of $p$, then $G$ is said to be a $p$-group.
