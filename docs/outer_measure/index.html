Outer Measure

\begin{definition}

An outer measure on a set $X$ is a function $\mu^\ast:\pow(X)\to[0,\infty]$ such that

  1. $\mu^\ast(\e)=0$.
  2. $\mu^\ast(A)\leq\mu^\ast(B)$ whenever $A\subeq B$.
  3. $\mu^\ast(\bigcup_{n<\omega}A_n)\leq\sum_{n<\omega}\mu^\ast(A_i)$ for any countable family $A_n\subeq X$.

\end{definition}

\begin{fact}[\ref[Construction of outer measures]{construction_of_outer_measures}]\label{construction}

Let $\mc{A}\subeq\pow(X)$ be a collection of subsets of $X$ containing $\e$ and covering $X$, and let $\mu:\mc{A}\to[0,\infty]$ be any function. Then $\mu^\ast:\pow(X)\to[0,\infty]$ defined by $\mu^\ast(S)\coloneqq\inf\sum_n\mu(A_n)$, where the infimum ranges over all $A_n\in\mc{A}$ such that $S\subeq\bigcup_nA_n$, is an outer measure on $X$.

\end{fact}

\begin{fact}[\ref[Extension to outer measures]{extension_of_premeasure_to_outer_measure}]

Let $\mu_0$ be a premeasure on an algebra $\mc{A}$. Then $\mu_0^\ast$ defined as in Fact \iref{construction} is an outer measure extending $\mu_0$.

\end{fact}
