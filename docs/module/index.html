Motivation & Definition

Throughout, fix a ring $R$.

Definition.

A (left) $R$-module is an

Abelian group $(M,+)$ equipped with a left

$R$-action $R\to\End_\catabgrp(M)$. That is, a function $R\times M\to M$ such that $$\begin{equation} r(m+n)=rm+rn,\ \ \ \ (r+s)m=rm+sm,\ \ \ \ (rs)m=r(sm),\ \ \ \ \mathit{and}\ \ \ \ 1m=m \end{equation}$$ for all $r,s\in R$ and $m,n\in M$.
