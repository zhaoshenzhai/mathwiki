Bases for Topologies

Motivation & Definition

Let $X$ be a set. To specify a topology on $X$, one needs to give a complete description of its open sets, but in practice, certain subcollections suffice. Two questions arise:

  1. If $\mc{T}$ is a given topology on $X$, what subcollections $\mc{B}\subseteq\mc{T}$ completely specify $\mc{T}$?
  2. What kinds of collections $\mc{B}\subseteq\pow(X)$ can specify some topology $\mc{T}$ on $X$?

In either case, we say that $\mc{T}$ is generated by $\mc{B}$.

Generation of a Fixed Topology

Let $\mc{T}$ be a topology on $X$.

Definition. A collection $\mc{B}\subseteq\mc{T}$ is a basis for $\mc{T}$ if every open set $U\in\mc{T}$ is a union of sets in $\mc{B}$.

Equivalently, $\mc{B}\subseteq\pow(X)$ is a basis for $\mc{T}$ iff for each $x\in X$, its neighborhood filter $\mc{N}_x$ (of all sets $A\subseteq X$ such that $x\in U\subseteq A$ for some $U\in\mc{T}$) admits a filter base in $\mc{B}$.

Proof.
Click to expand...
If $\mc{B}\subseteq\mc{T}$ is a basis and $x\in X$, then $\l\{V\in\mc{B}\st x\in V\r\}$ is a filter base for $\mc{N}_x$. Conversely, let $U$ be open. Every $x\in U$ admits a filter base $\mc{B}_x\subseteq\mc{B}$, giving us for each $x\in U$ some $B_x\in\mc{B}_x\subseteq\mc{B}$ with $x\in B_x\subseteq U$, so that $U=\bigcup_{x\in U}B_x$.$\blacksquare$

Since all open sets $U\in\mc{T}$ are just unions of sets in $\mc{B}$, we call $\mc{B}$ the basic open sets.


  Thus $\mc{T}$ is obtained from $\mc{B}$ by closing under unions. To further thin down the subcollection and obtain $\mc{T}$ by closing under finite intersections as well, we need:

Definition. A collection $\mc{S}\subseteq\mc{T}$ is a subbasis for $\mc{T}$ if every open set $U\in\mc{T}$ is a union of finite intersections of sets in $\mc{S}$.

Note that $\mc{B}_\mc{S}\coloneqq\l\{\bigcap_{i=1}^nU_i\st U_i\in\mc{S},n\in\N\r\}$ is a basis for $\mc{T}$.

Generation of Some Topology

One way for a collection $\mc{B}\subseteq\pow(X)$ of subsets of $X$ to generate some topology on $X$ is to let it satisfy some of the axioms for a topology on $X$, from which a topology can be obtained by closing it under the others. To illustrate, consider the following

Definition. A collection $\mc{B}\subseteq\pow(X)$ is a (saturated) basis for a topology if $X$ and all pairwise intersections of sets in $\mc{B}$ are unions of sets in $\mc{B}$ (resp. in $\mc{B}$).

In other words, $\mc{B}$ is a basis if it covers $X$ and for every $B_1,B_2\in\mc{B}$ with $x\in B_1\cap B_2$, we have $x\in B\subseteq B_1\cap B_2$ for some $B\in\mc{B}$. In either case, we obtain a topology on $X$ by closing it under unions, i.e. by letting $$\begin{equation} \mc{T}_\mc{B}\coloneqq\l\{U\subseteq X\st\fa x\in U,\ex B\in\mc{B},x\in B\subseteq U\r\}, \end{equation}$$ in which case $\mc{B}$ is a basis for $\mc{T}_\mc{B}$.

Proof.
Click to expand...

Since every saturated basis is a basis, we prove it if $\mc{B}$ is just a basis. That $\em\in\mc{T}_\mc{B}$ is vacuous, and $X\in\mc{T}_\mc{B}$ since $\mc{B}$ covers $X$. Closure under arbitrary unions is clear. If $U_1,U_2\in\mc{T}_\mc{B}$ and $x\in U_1\cap U_2$, then $x\in B_i\subseteq U_i$ for $i=1,2$ for some $B_1,B_2\in\mc{B}$, and so $x\in B\subseteq B_1\cap B_2\subseteq U_1\cap U_2$ for some $B\in\mc{B}$.


  Finally, it is clear that $\mc{B}$ is a basis for $\mc{T}_\mc{B}$ since every $U\in\mc{T}$ can be written as $U=\bigcup_{x\in U}B_x$, where each $B_x\in\mc{B}$ is such that $x\in B_x\subseteq U$.$\blacksquare$

One can be more vigorous by removing all the axioms of a topology and consider any collection $\mc{S}\subseteq\pow(X)$. This induces a basis $\mc{B}_\mc{S}$ consisting of finite intersections of sets in $\mc{S}$, from which we obtain a topology $\mc{T}_\mc{S}\coloneqq\mc{T}_{\mc{B}_\mc{S}}$ with $\mc{S}$ as a subbasis thereof.

Proof.
Click to expand...
We show that $\mc{B}_\mc{S}\coloneqq\l\{\bigcap_{i<n}U_i\st U_i\in\mc{S}\r\}$ is a saturated basis. Indeed, that $X\in\mc{B}_\mc{S}$ follows from letting $n=0$. That $\mc{B}_\mc{S}$ is closed under pairwise intersections is clear, since $\bigcap_{i<n}U_i\cap\bigcap_{i<n’}U’_i$ is still a finite intersection.$\blacksquare$

Rigidity of Bases/Subbases

Many topological properties of $X$ can be detected by bases/subbases thereof.
