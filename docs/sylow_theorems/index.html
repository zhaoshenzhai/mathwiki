The Sylow Theorems

Motivation & Statements

How many subgroups does a finite group $G$ have? And can we classify them, in order to study $G$ itself? Well, Lagrange’s Theorem imposes that every subgroup $H\substructeq G$ must have order dividing $|G|$, so this limits their possibilities. A natural question to ask at this point is whether this is the only restriction, that is, if $d$ divides $|G|$, does there necessarily exist a subgroup $H\substructeq G$ of order $G$? Alas, this is not true, since $G\coloneqq A_4$ has no subgroup of order $6$.
  However, that $\Z$ is a UFD suggests writing $|G|=p^rm$ for some prime $p$ dividing $|G|$ and $m\perp p$, so we can focus on the existence of subgroups one prime at a time. Then, it turns out that there is a subgroup $S_p\substructeq G$ with order $p^r$, called a $p$-Sylow subgroup of $G$, and we a lot of control over their existence (Sylow I), their structure (Sylow II), and the number of such subgroups (Sylow III):

Theorem (The Sylow Theorems).

Let $G$ be a finite group and write $|G|=p^rm$ for some prime $p$ with $m\perp p$.

  1. (Sylow I). $G$ contains a $p$-Sylow subgroup $S_p$ and every $p$-subgroup $H\substructeq G$ extends to a $p$-Sylow subgroup.
  2. (Sylow II). All $p$-Sylow subgroups of $G$ are conjugate.
  3. (Sylow III). The number $n_p$ of all $p$-Sylow subgroups of $G$ divides $m$ and is congruent to $1$ modulo $p$.

This is nice! It gives precise quantitative control over a large class of subgroups of $G$, which is a major step towards understanding the structure of $G$ itself. For instance, if we know via Sylow III that $G$ admits a unique $p$-Sylow subgroup $P$, then it is automatically normal by Sylow II, and this allows us to form the quotient $G/P$, which has order $m<|G|$.

Proof of the Sylow Theorems

Essentially, the proof of all three Sylow Theorems boil down to careful application of the ‘fixed point theorems for $p$-groups’, which is extremely simple: if we have an action $\phi:P\act X$ of a $p$-group on a finite set, then $|Z|\equiv|X|\,\mod\,p$ where $Z$ is the fixed points of $\phi$.

Cauchy’s Theorem

Subgroups $P\substructeq G$ with order $p^s$ for some $s\leq r$ are called the $p$-subgroups of $G$, so we need to understand their structure. The fundamental lemma that drives Sylow I is that they exist when $s=1$:

Theorem (Cauchy). If $G$ is a finite group and $p$ is a prime divisor of $|G|$, then $G$ contains an element of order $p$.

We give two proofs. The first illustrates the ‘abelian $\rightsquigarrow$ center’ technique, and the second is an application of the fixed point theorems.

Proof (abelian $\rightsquigarrow$ center). Click to expand... First, consider the case when $G$ is abelian. We proceed by induction on $|G|$, so let $H\substruct G$ be maximal. If $p$ divides $|H|$, then we are done by induction, so suppose otherwise and set $K\coloneqq\gen{x}$. Since $G$ is abelian, we have $HK=KH$, and hence we have$^\dagger$ $H\substruct HK\substructeq G$. Maximality of $H$ then forces $HK=G$, so $|G|=|HK|=|H||K|/|H\cap K|$. Thus $p$ divides $|H||K|$, so $p$ divides $|K|=\ord x$. Writing $\ord x=pk$ for some $k\in\N$, we see that $\ord x^k=p$ is as desired.
  In general, consider the abelian subgroup $Z(G)\substructeq G$. If $p$ divides $|Z(G)|$, we are done. Otherwise, by the Class Equation we have $$\begin{equation} |G|=|Z(G)|+\sum_{a\in A}[G:Z_G(a)], \end{equation}$$ where $A\subseteq G$ is a transversal of non-trivial conjugacy classes. Since $p$ divides $|G|$ but not $|Z(G)|$, this furnishes some $a\in A$ such that $p\ndivides[G:Z_G(a)]$, so $p$ divides $|Z_G(a)|$. Finally, since $a\not\in Z(G)$, the subgroup $Z_G(a)\substruct G$ is proper, so we are done by induction.$\blacksquare$
Proof (fixed point theorem). Click to expand... Let $S\subseteq G^p$ be the subset of tuples $(x_1,\dots,x_p)$ such that $x_1\cdots x_p=e$. Then $|S|=|G|^{p-1}$, so $p$ divides $|S|$. Moreover, $S$ is closed under cyclic permutations, so we have an action $\phi:\Z/p\Z\act S$ generated by $1\cdot(x_1,\dots,x_p)\coloneqq(x_2,\dots,x_p,x_1)$.
  Observe that $\Z/p\Z$ is a $p$-group, so$^\dagger$ $|Z|\equiv|S|\,\mod\,p$ where $Z$ is the fixed-points of $\phi$. Thus $p\geq2$ divides $|Z|$ too, so $|Z|>1$. But elements in $Z$ are $(x,\dots,x)\in S$ for $x\in G$, so there is some non-trivial $x\in G$ with $x^p=e$, as desired.$\blacksquare$

Tower of $p$-subgroups

We now construct a tower of $p$-subgroups with composition factors of order $p$, so with the existence of the base case, Sylow I follows by repeated application of the following Proposition. First, we need a lemma.

Lemma. If $H\substructeq G$ is a $p$-subgroup, then$^\dagger$ $[N_G(H):H]\equiv[G:H]\,\mod\,p$.
Proof. Click to expand... We are done if $H$ is trivial. Otherwise, $H$ acts on $G/H$ by left-multiplication, whose fixed-points are the cosets $gH$ such that $h(gH)=gH$ for all $h\in H$, which are those such that $H\subseteq g^{-1}Hg$ $-$ or equivalently, $g\in N_G(H)$. Thus, the fixed-points thereof are the cosets $gH$ of $H$ where $g\in N_G(H)$, of which there are $[N_G(H):H]$ many. Now, apply the fixed point theorem.$\blacksquare$
Proposition. If a $p$-subgroup $H\substructeq G$ is not $p$-Sylow, then there is a $p$-subgroup $K\substructeq G$ such that $H\nsubgrpeq K$ and $[K:H]=p$.

Proof. Click to expand... Since $H$ is not a $p$-Sylow subgroup, we see that $p$ divides $[G:H]$, and hence $p$ divides $[N_G(H):H]$ too. By Cauchy’s Theorem, there is an element in $N_G(H)/H$ of order $p$, which generates a cyclic subgroup $K_0$ with $|K_0|=p$. Set $K\coloneqq\pi^{-1}(K_0)\substructeq G$, which clearly contains $H$. Since $\pi(H)\nsubgrpeq\pi(K)$, we have by the Lattice Isomorphism Theorem that $H\nsubgrpeq K$ and $K/H\iso\pi(K)/\pi(H)\iso K_0$, and thus $[K:H]=|K_0|=p$ as desired.$\blacksquare$

Structure of $p$-Sylow subgroups

Given that a $p$-Sylow subgroup $S_p\substructeq G$ exists, the rest is just combinatorics.

Proof (of Sylow II). Click to expand... We need to show that if $P$ is a $p$-Sylow subgroup, then $P=gS_pg^{-1}$ for some $g\in G$. Indeed, let $P$ act on $G/S_p$ by left-multiplication, so there are$^\dagger$ $|G/S_p|=[G:S_p]$ number of fixed points modulo $p$. But $[G:S_p]=m\perp p$, so this action admits a fixed point $gS_p$ for some $g\in G$. Hence $g^{-1}Pg\substructeq S_p$, so $P=gS_pg^{-1}$ by order considerations.$\blacksquare$
Proof (of Sylow III). Click to expand... Note that $n_p$ is the cardinality of the conjugacy class of $S_p$, which by Orbit-Stabilizer is $[G:N_G(S_p)]$, so we have by Lagrange’s Theorem that $m=[G:S_p]=n_p[N_G(S_p):S_p]$. Thus $n_p\divides m$. Furthermore, we have by the Lemma that $m$ is congruent to $[N_G(S_p):S_p]$ modulo $p$, so $mn_p\equiv m\,\mod\,p$. As $m\perp p$ and $p$ is prime, this gives $n_p\equiv1\,\mod\,p$ as desired.$\blacksquare$
