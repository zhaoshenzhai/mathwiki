$p\ $-group

By Lagrange’s Theorem, if a finite group $G$ has prime order $p$, then $G\iso\,$ $\Z/p\Z$ and has no non-trivial subgroups. It is natural to ask for a generalization when $\l|G\r|$ is a power of $p$, or more generally, when the order of every $g\in G$ is a power of $p$.


  Throughout, fix a prime $p$.

Finite $p$-groups

If $G$ is a finite group, then the condition that the order of every $g\in G$ is a power of $p$ is $-$ by Lagrange’s Theorem $-$ equivalent to the following

Definition. A $p$-group is a group $G$ with $\l|G\r|=p^n$ for some $n\in\N$.
